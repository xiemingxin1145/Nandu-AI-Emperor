## 二、序章：明道宫惊醒

### 历史背景说明（供开发参考）

```
建炎元年五月初一（1127年6月12日）
康王赵构于南京应天府鸿庆宫祭祀赵宋祖庙后，正式即位为帝，改元建炎。
彼时，二帝北狩，汴京沦陷，中原已失，朝廷仅余残兵败将南聚。
主战的宗泽在开封艰难守御，主和的汪伯彦、黄潜善已在耳边谗言。
李纲虽被任为首相，却已是风中残烛。
```

**游戏设定**：玩家（穿越者）在即位大典当夜，于应天府偏殿"明道宫"（取"明天下之道"之意，为鸿庆宫附属礼仪殿宇）中"苏醒"，意识到自己已穿越进入赵构之身。

---

### 序章分镜（文字描述版，供CG参考）

#### 幕一：入夜（黑屏字幕）

```
"靖康板荡，中原陆沉。"
"建炎元年，五月。"
"天下，只剩这一个姓赵的还活着。"
```

#### 幕二：大典散场（CG 01：南渡夜境）

*场景*：应天府外，夜色中南逃的百姓车马、疲惫的禁军、远处地平线上昏红的火光。  
*氛围*：不是胜利，是劫后余生。战旗猎猎，却没有人敢直视北方。

#### 幕三：偏殿独处（CG 02：明道宫惊醒）

*场景*：殿内烛火将熄，玉案上搁着还未盖印的诏书，一个人独坐，周遭空无一人。  
*文本*：

> 烛芯噼啪一声。  
> 眼前的朱砂诏书、摇晃的宫灯、手腕上勒着的明黄衣袍——
> 都不对。  
> 都不是自己的。  
> 但心跳声是真实的，手上的冷汗是真实的，窗外传进来的"万岁"余音是真实的。  
>   
> 他坐在这里。  
> 今天，他成了皇帝。  
> 明天，他不知道会不会死。

#### 幕四：第一个人进来（NPC 初登场）

宫门轻叩三声。内侍入殿，带来的不是贺词——是宗泽从开封发来的急奏：

> "金人再集兵马，开封守军已不足两万，臣请陛下御驾亲征，渡河收复！"

---

### 序章第一道抉择

> **【第一抉择】宗泽的急奏，你怎么回？**

```
A. 批复"知道了，命宗泽坚守，朕稍后议"
   → 温吞，宗泽寒心，但局面稳住
   → 解锁提示：「汪伯彦和黄潜善会来找你的」

B. 当场写诏：调岳飞所部北上策应宗泽
   → 强硬，宗泽大喜，岳飞早早进入视野
   → 解锁提示：「汪伯彦和黄潜善会来劝你收回」

C. 召李纲入殿，当面议此事
   → 稳健，李纲信任+1，主战派士气+
   → 解锁提示：「李纲只剩不多的日子了——趁现在」
```

三个选项无对错，只有后果。游戏第一句话就要让玩家明白这一点。

## 四、前三回合新手引导

### 设计原则

- **不堆系统**：一回合只开一个功能区
- **每回合有一个真实后果**：玩家的选择要在当回合就看到反馈
- **引导用NPC对话完成**，不用教学弹窗

---

### 第一回合：建炎元年五月（称帝后第一天）

**开放系统**：
- ✅ 朝议（限定：只有三位大臣登场）
- ✅ 人物（只显示已登场的NPC）
- ✅ 奏折（只有1封：宗泽急奏）
- ❌ 军务（锁定）
- ❌ 城池（锁定）
- ❌ 地图（锁定）
- ❌ 外交（锁定）

**剧情事件**：

1. 宗泽急奏送达（已在序章触发）
2. 李纲入朝，陈述整军方略（一席话 = 教玩家看人物关系面板）
3. 汪伯彦旁敲侧击，建议"先稳住，不宜操之过急"（第一次感受到朝廷内部拉力）

**本回合核心抉择**：

> 【是否批准李纲的整军奏请？】  
> 批准 → 李纲信任+2，军务系统下回合提前开放  
> 拒绝 → 汪伯彦好感+1，但埋下李纲被架空伏笔

**回合结尾提示**（内侍念白）：
> "陛下，今日朝议已毕。天色渐晚，明日还有诸多政务待理。"  
> *（解锁提示：下回合军务系统将开放）*

---

### 第二回合：建炎元年六月（称帝后一个月）

**开放系统**：
- ✅ 军务（新增，教程引导）
- ✅ 城池（基础，只显示应天府一城）
- ✅ 地图（仅显示可视区域，大部分为迷雾）

**剧情事件**：

1. 宗泽来信，报告开封守御战况（第一次看到战况面板）
2. 军务教程：派一支小队巡查江淮防线（实际上是手把手教"调兵"操作）
3. **关键事件**：李纲建议——派岳飞所部扼守滑州，配合宗泽

**本回合核心抉择**：

> 【李纲 vs 汪伯彦 的第一次正面冲突】  
> 李纲：派兵北上，策应宗泽  
> 汪伯彦：保留兵力，先守扬州行宫准备  
>   
> 这道题没有标准答案，但选择汪伯彦的玩家，下一回合李纲会主动请辞。

**新NPC登场**：岳飞（以一封请战书形式出现，还没有立绘，只有名字）

---

### 第三回合：建炎元年七月（多事之秋）

**开放系统**：
- ✅ 外交（基础，金使来书）
- ✅ 派系面板（正式开放，显示主战/主和分布）
- ✅ 国策（第一条国策可选）

**剧情事件**：

1. 金国来书，措辞傲慢，要求割地称臣
2. 派系面板教程：李纲（主战）vs 汪伯彦/黄潜善（主和）
3. **核心爽点**：若前两回合护住了李纲，此时李纲当庭拒绝金使，全场哗然

**本回合核心抉择**：

> 【金使的国书，怎么回？】  
> A. 当众撕书，宣布不议和  
>   → 主战派士气暴涨，金人震怒，但全体武将归心  
> B. 留而不答，托辞需"商议"  
>   → 给自己留余地，但群臣开始揣摩圣意  
> C. 密传李纲：你去回绝，不要扯上朕  
>   → 最机巧，但李纲独自承压，忠诚度小降

**第三回合结束后**：
> 派系系统、国策系统全面开放。新手引导结束。  
> 屏幕提示："山河虽破，尚可为也。"

## 五、第一批核心NPC出场顺序与角色设定

### 出场顺序

| 出场时机 | NPC | 出场方式 |
|---|---|---|
| 序章 | **内侍总管**（虚构，无名） | 第一个进殿的人，带来宗泽奏折 |
| 第一回合 | **李纲** | 正式朝议，慷慨陈词 |
| 第一回合 | **汪伯彦** | 朝议后私下进言，笑脸相迎 |
| 第一回合 | **黄潜善** | 随汪伯彦出场，一语双关 |
| 第二回合 | **岳飞**（仅名字/书信） | 请战书，无立绘，只有文字 |
| 第二回合 | **宗泽**（来信） | 战况急报，不在应天府 |
| 第三回合 | **韩世忠** | 金使来书后，武将代表发言 |
| 第三回合末 | **赵鼎**（初现） | 以御史身份旁观，寥寥一句话 |

---

### 核心NPC角色卡（第一批）

#### 李纲
```
立场：主战
性格：刚直，不善变通，说话像上奏疏
初始关系：中立偏好（知道你刚登基，给机会）
专属事件：「李纲去职」——若三回合内被汪伯彦架空，第四回合必然辞相
历史结局钩子：真实历史中他只做了75天首相，玩家能改变吗？
台词风格：半文半白，惜字如金，从不说废话
```

#### 汪伯彦
```
立场：主和/保皇派（实为保自己）
性格：笑面佛，话里话外都是"为陛下着想"
初始关系：偏好（帮助你称帝有功）
专属事件：「汪公献策」——每当玩家犹豫时他必然出现，提供"最安全"的选项
台词风格：絮絮叨叨，话多，总说"臣以为……然则……"
```

#### 宗泽
```
立场：主战（最强硬）
性格：倔，老，不会妥协
初始关系：尊重（他是真的忠臣，摆明了来找你托付）
专属事件：「三呼渡河」——历史中宗泽临死前三次大呼"渡河！"，玩家能在他死前做什么？
台词风格：急促，直接，永远只说打仗的事
```

#### 岳飞
```
立场：主战（纯粹武人）
性格：话少，行多，忠诚但认死理
初始状态：无立绘，只有名字（建炎元年他才25岁，还未成名）
解锁条件：第二回合后，批准他的请战书，立绘解锁
专属剧情线：从小卒到大将的完整成长线，含「风波亭」历史节点
```

#### 韩世忠
```
立场：主战，但比岳飞圆滑
性格：豪爽，市井气，打仗不要命
初始关系：中立（他认皇帝，不认你这个人）
专属事件：「黄天荡之战」——历史中以八千兵困金兀术十天
```

#### 赵鼎
```
立场：主战且主政，政治头脑远超李纲
性格：沉稳，话少，看人极准
初始关系：观望（他在等你值不值得辅佐）
专属事件：「赵相回朝」——若护住李纲失败，赵鼎是主战派的后续顶梁
```

## 七、第一阶段主线目标（建炎元年——建炎二年）

### 阶段一目标结构

```
【明线目标】
稳住朝廷，建立最初的权力班底

【暗线目标】
在"主战-主和"的派系撕裂之前，
先让一两个关键人物对你真正死心塌地
```

### 主线节点（按时间顺序）

#### 节点 1：「李纲存废」
```
时间：建炎元年六月——七月
问题：历史上李纲只做了75天首相就被汪伯彦架空赶走
机会点：玩家若在第一回合明确支持李纲，可以延长他的任期
成功结果：主战派在第一阶段不崩盘
失败结果：李纲去职，汪伯彦/黄潜善主持朝政，后续主战线难度大增
```

#### 节点 2：「宗泽的最后」
```
时间：建炎二年七月
问题：宗泽守开封已到极限，本月将死，临终三呼"渡河！"
机会点：在他死前，能否送去一道有重量的诏书，让他死而无憾？
         能否让岳飞在他死之前抵达，延续他的意志？
成功结果：宗泽系武将（包括岳飞早期班底）士气保住
失败结果：开封系武将群龙无首，后续必出一个"杜充"祸事
```

#### 节点 3：「扬州行宫之争」
```
时间：建炎元年末
问题：朝廷要不要迁往扬州？
      汪伯彦推荐扬州——历史上这是个大坑（建炎三年在扬州被金兵突袭）
机会点：若玩家识破这个坑，选择留守应天或南移江宁
成功结果：建炎三年金兵奔袭扬州时，朝廷不在那里
失败结果：建炎三年狼狈渡江，龙袍玉玺都丢了（历史原样重演）
```

#### 节点 4：「第一批武将收心」
```
时间：建炎元年——建炎二年
目标：在主线第一阶段结束前，至少让以下一位武将与你建立真正的信任：
      岳飞 / 韩世忠 / 刘光世（选一）
机会点：各有专属事件，各有不同的收心条件
重要性：武将信任不够，绍兴年间就管不住他们
```

## 九、备注：设计红线

1. **不要让玩家第一回合就感到必败**——每一道抉择都要有可以"打赢"的选项
2. **配角必须有自己的逻辑**——李纲不是工具人，他有自己想要的，有会让他寒心的事
3. **历史事件不直接"告知"玩家**——宗泽会死这件事，游戏里只给征兆，不给剧透
4. **派系拉力是持续的**——汪伯彦不是反派boss，他是一个真实的力量，会一直在
5. **穿越感要轻**——玩家知道自己穿越了，但游戏本体不大量强调这一点，重心在"你现在怎么办"
