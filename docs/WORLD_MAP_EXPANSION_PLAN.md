# 《南渡无悔》世界地图扩展规划

## 核心原则

当前版本不要一次性塞满全中国地图。先把世界地图分成大区，占坑规划，后续按版本逐步展开。

目标不是单纯多城，而是让玩家看见：南宋、金国、西夏、大理、吐蕃诸部、海贸势力共同存在于一个更大的世界里。

## 分阶段落地

### Phase 1：当前可玩主战区

优先保证南宋和金国主线闭环。

- 江南行在区：临安、建康、镇江、平江、明州、绍兴
- 江淮防线区：扬州、楚州、庐州、寿春、泗州、濠州
- 荆襄山河区：襄阳、鄂州、江陵、随州、郢州
- 中原旧都区：开封、洛阳、应天、郑州、许州、陈州

玩法重点：皇宫、城池经营、淮河防线、北伐、金军反扑、光复东京。

### Phase 2：下一阶段扩展区

增加地图纵深和第三势力，不急着全做细。

- 河北燕云区：燕京、中山、真定、河间、大名、云中
- 西夏河陇区：兴庆府、灵州、夏州、凉州、兰州、会州

玩法重点：金国纵深、燕云屏障、西夏外交、买马、边贸、联夏制金。

### Phase 3：远期世界地图区

先作为地图边缘和事件来源，不作为前期主战场。

- 大理西南区：大理、善阐、姚州、永昌
- 吐蕃西域区：河湟、西宁、于阗、龟兹
- 东海海贸区：明州、泉州、广州、琉球、高丽海路

玩法重点：海贸、边疆外交、马政、商道、宗教、海外收益。

## 后续版本建议

### V1.3：世界地图大区 UI

- 地图上显示大区标签：江南、江淮、荆襄、中原、河北、西夏、大理、吐蕃、东海
- 增加大区快速定位按钮
- 先不新增大量城市，只显示区域存在感

### V1.4：西夏占坑

- 新增西夏势力 ID
- 新增兴庆府、灵州、夏州等少量节点
- 西夏不立即大规模参战，先做外交事件和贸易收益

### V1.5：河北燕云纵深

- 加燕京、真定、中山、河间、云中
- 金国 AI 有后方恢复和调兵逻辑
- 北伐不再只打到开封就结束

### V1.6：海贸与大理远区

- 港口城市获得海贸收益
- 明州、泉州、广州成为海贸节点
- 大理作为外交和资源事件来源

## 城池扩展规则

每次新增城池不要超过 6 到 8 个。

每个新增城池必须同时补三件事：

1. MapData 节点坐标
2. road/route 连接
3. City 初始数据与势力归属

不要只加城市名，否则地图会变成一堆没有玩法的点。

## 地图视觉规则

大地图要能一眼看出：

- 长江、黄河、淮河
- 太行山、秦岭、河陇山地
- 江南水网
- 西夏河套与河西走廊
- 燕云北方屏障
- 东南海贸线

先做 2D/2.5D 可读地图，不做真 3D。
