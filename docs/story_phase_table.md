# 《南渡无悔》剧情阶段表

## 第一章·南渡之始（建炎元年 正月～三月）
**核心压力：** 立足未稳、朝堂分裂、金兵压境  
**核心任务：** 稳住人心、确立权威、寻访人才  

| 回合 | 阶段 | 核心事件 | 主要矛盾 |
|------|------|----------|----------|
| 1 | 即位初朝 | 建炎初议（必触发） | 主战vs主和 |
| 2-3 | 人才浮现 | 宗泽北望、韩世忠请战 | 是否给资源 |
| 3-5 | 人才寻访 | 军中发现岳飞 | 是否拔擢无名之辈 |
| 5-8 | 财政危机 | 国库告罄 | 如何筹钱 |
| 6-10 | 派系争斗 | 主和疏议、李纲去位 | 主战派能否守住 |

**章节结束条件：** 朝廷稳定度>30 且 至少有2名武将被任命

---

## 第二章·江淮震动（建炎元年 四月～建炎二年 二月）
**核心压力：** 金军南下、粮草告急、民心动摇  
**核心任务：** 构建江淮防线、稳定后方  

| 回合 | 阶段 | 核心事件 | 主要矛盾 |
|------|------|----------|----------|
| 8-12 | 金兵渡淮 | 金兵渡淮（必触发） | 守还是退 |
| 10-15 | 城池危机 | 扬州惊变、建康围城 | 资源如何分配 |
| 8-20 | 后方压力 | 粮草告急、军中疫病 | 内政vs军务 |
| 8-20 | 义军问题 | 义军蜂起 | 招抚还是放任 |
| 10-20 | 民心动摇 | 天子南逃流言 | 形象建设 |

**章节结束条件：** 金威胁度下降至40以下 或 建康防御度>30

---

## 第三章·黄天荡前（建炎二年 三月～建炎四年）
**核心压力：** 完颜宗弼大举南征、秦桧回朝  
**核心任务：** 黄天荡决战、防止奸臣得势  

| 回合 | 阶段 | 核心事件 | 主要矛盾 |
|------|------|----------|----------|
| 15-25 | 兀术南征 | 完颜宗弼南征（必触发） | 全力迎战or求和 |
| 18-30 | 奸臣回朝 | 秦桧归来 | 信任or监视 |
| 20-35 | 北伐争议 | 张浚北伐议、岳飞声望崛起 | 激进北伐or稳健守御 |
| 15-30 | 民间声望 | 背刺精忠传说 | 武将声望管理 |

**章节结束条件：** 完成黄天荡事件 且 朝局稳定

## 人物成长轨迹（参考）

### 岳飞
```
军中小卒（开局）→ 百夫长（3-5回合）→ 偏将（8-12回合）
→ 统领（15-20回合）→ 方面大将（25+回合）
```
**可统兵上限：** 500 → 2000 → 10000 → 30000 → 80000+

### 韩世忠  
```
低阶将领（开局）→ 江防统领（2-6回合）→ 水军都督（10+回合）
```
**水战专精：** 黄天荡加成+30%

### 秦桧
```
寒门文士（开局，主和倾向）→ 归正人（18-30回合）
→ 主和核心（若获重用）→ 朝堂毒瘤（30+回合，若不加约束）
```
