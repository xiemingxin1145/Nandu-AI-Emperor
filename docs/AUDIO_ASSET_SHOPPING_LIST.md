# 音频素材清单 v1

本清单用于给 Claude/Cursor 或素材整理工具批量寻找、导入音频。

## 目录约定

```text
app/src/main/assets/audio/bgm/
app/src/main/assets/audio/ui/
app/src/main/assets/audio/sfx/
app/src/main/assets/audio/ambience/
app/src/main/assets/audio/voice/
```

```text
bgm_main_menu.ogg        主菜单：古琴、箫、低鼓，皇权感
bgm_palace_hall.ogg      皇宫大厅：庄重、宽阔、轻鼓
bgm_court.ogg            垂拱殿听政：低沉、肃穆
bgm_court_council.ogg    朝会裁断：压迫、紧张但不吵
bgm_map.ogg              山河地图：大地图、战略感
bgm_city.ogg             城池经营：轻松、市井、内政
bgm_market.ogg           市舶/商贸：热闹、商船、轻快
bgm_diplomacy.ogg        外交：肃穆、异域、礼仪感
bgm_military.ogg         军务：鼓点、号角、紧迫
bgm_crisis.ogg           危机：低音、悬疑、压迫
bgm_event_sad.ogg        失利/悲情：低缓、悲凉

bgm_victory.ogg          胜利：凯旋、恢宏

```
