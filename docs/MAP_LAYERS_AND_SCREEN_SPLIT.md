# 山河图层与页面职责拆分

本文件用于修正“山河、军务、国政内容重叠”的问题。

## 当前问题

旧版本中，山河页同时承担：

- 地图浏览
- 城池操作
- 军团显示
- 前线显示
- 部分经营入口
- 部分外交/外贸提示

这会导致玩家感觉所有内容挤在一张图上，军务与国政也互相重叠。

## PR #7 已接入

山河页新增四个图层：

1. 军事
2. 经济
3. 外交
4. 外贸

### 军事图层

定位：看战局。

显示重点：

- 宋金城池
- 前线光效
- 军团旗帜
- 行军路线
- 关隘道路

不重点显示：

- 港市收益
- 外交势力关系
- 海贸利润

- 市舶司收入

## 页面职责定盘

### 皇宫

总入口。看八宫殿待办、进入大殿朝会、处理宫殿事件。

### 朝议 / 垂拱殿

处理圣旨、AI 解析、群臣奏议、皇帝最终裁断。

### 山河

只负责“看天下”：

- 地图
- 城池
- 线路
- 势力范围
- 军事/经济/外交/外贸图层

山河可以保留城市详情和快捷拟旨，但不应该成为完整军务或完整国政面板。

### 国政 / 政事堂

只负责“管钱粮民心”：

- 国库
- 粮草
- 民心
- 城池经济
- 漕运
- 市舶司
- 外贸收入

### 军务 / 枢密院

只负责“管军队”：

- 军团
- 将领
- 战报
- 调兵
- 粮道
- 训练
- 攻守计划

### 御书房

只负责“看密折和复盘”：

- 密折
- 传闻
- 外交情报
- 前次选择后果
- 风险提示
